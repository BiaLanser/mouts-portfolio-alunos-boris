Aprendendo HTML

Quer ver como é simples aprender HTML? Vamos nessa!

Listas

No HTML, podemos criar listas ordenadas e não ordenadas

Exemplo de lista ordenada:

  1. Item 1
  2. Item 2
  3. Item 3

Para fazer, iniciamos com a tag <ol> e dentro dela a tag <li> para cada item

Exemplo de lista não ordenada:

  • Item 1
  • Item 2
  • Item 3

Para fazer, iniciamos com a tag <ul> e dentro dela a tag <li> para cada item

Inserindo imagens e links

No HTML, podemos inserir imagens e links

Exemplo de imagem:

Para fazer, utilizamos a tag <img> e, dentro dela, definimos o caminho da imagem com o atributo src

No HTML, é possível utilizar links

Exemplo de link:

Gerar padrões de armadura no Minecraft

Para fazer, utilizamos a tag <a> e, dentro dela, definimos o caminho do link com o atributo href

Tabelas

No HTML, podemos criar tabelas

Exemplo de tabela:

Coluna 1 Coluna 2
Dado 1 Dado 2
Dado 3 Dado 4

Para fazer, iniciamos com a tag <table> e dentro dela a tag <tr> para cada linha e a tag <td> para cada coluna
A tag <th> é utilizada para definir o cabecalho, como exibido no exemplo em Coluna 1 e Coluna 2
